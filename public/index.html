Quick Guide
Getting AppState
  1. Login to Facebook in your browser
  2. Press F12 for Developer Tools
  3. Go to Application/Storage tab
  4. Copy cookies from facebook.com
  5. Convert to JSON array format
Starting a Boost
  1. Paste your AppState JSON
  2. Enter Facebook post URL
  3. Set number of shares (optional)
  4. Use 1-second delay for speed
  5. Click "Start Boost" button
Important Notes
  • 1-second delay recommended
  • Unlimited mode has no restrictions
  • Monitor live processes
  • Stop boosts if needed
  • Cookies expire after some time
